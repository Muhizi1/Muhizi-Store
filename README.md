Muhizi Store is a modern e-commerce website designed for trading shoes ,rings, clothes, and cups.
It provides a smooth and user-friendly shopping experience with product browsing, search functionality, cart management, and user account features.

 Features
 Shopping Experience

Browse products (Shoes, Clothes, Cups)

Add products to cart

Remove products from cart

View total cart items

 Product Search

Search products by name

Quick filtering system

Real-time search results

 Navigation Menu

Home – Landing page

Shop – View all available products

Cart – Manage selected products

Profile Icon – Access user details

Logout – Secure logout functionality

 User Account

User profile access

Login / Logout system

Personalized shopping experience
